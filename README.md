<h2 align="center">Sistema Cadastro Usuarios com Prazo de Validade Mysql Python</h2>
<h3>Sistema de cadastro de usuários com nivel de administrador e usuario, cria do zero tabelas em python com codigo limpo pequeno e comentado para ajudar a comunidade, nunca foi tão facil entender como é criado um banco de dados, uma tabela e a inserção de dados nesta tabela, com este codigo seus sistemas ficarão mais seguros pois este codigo conta com sistema que irá fazer com que seu programa expire em data determinada por você e caso mude esta data tudo acontecerá em tempo real online permitindo assim renovar licenças de programas e criar usuários remotamente e controlar todos seus dados e fluxos!</p>
